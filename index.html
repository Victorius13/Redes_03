Fundamentos de Redes

Conceptos Básicos de Redes

IPv4

  • Dirección de 32 bits (4 bytes)
  • Formato decimal: 192.168.1.1
  • Rango: 0.0.0.0 a 255.255.255.255
  • Approx 4.3 mil millones de direcciones

IPv6

  • Dirección de 128 bits (16 bytes)
  • Formato hexadecimal: 2001:0db8:85a3:0000:0000:8a2e:0370:7334
  • Capacidad: 3.4×10³⁸ direcciones
  • Mejor seguridad y autoconfiguración

Tipos de direcciones IP

  • Unicast: Dirección única para un dispositivo
  • Broadcast: Envía a todos los dispositivos en la red
  • Multicast: Envía a un grupo específico de dispositivos

Conversión Binario-Decimal

Ejemplo: 1101.1011
Cálculo:
1*2³ + 1*2² + 0*2¹ + 1*2⁰ . 1*2⁻¹ + 0*2⁻² + 1*2⁻³ + 1*2⁻⁴
= 8 + 4 + 0 + 1 = 13
= 0.5 + 0 + 0.125 + 0.0625 = 0.6875
Resultado: 13.6875

Suma Binaria

  1 1 1  (acarreos)
    1 0 1 1  (11)
  +   1 1 1  (7)
  ---------
  1 0 0 1 0  (18)

Reglas: 0+0=0, 0+1=1, 1+1=0 con acarreo 1
